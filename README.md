# UCTS (Upcycled Ceramic Thermal Storage)

[ucts.solutions](https://ucts.solutions/)

Инновационное решение для хранения энергии, которое превращает промышленные отходы в доступные системы накопления тепловой энергии.

## О проекте

Этот репозиторий содержит полнофункциональное веб-приложение для UCTS, представляющее собой сайт-презентацию (pitch deck) для инвесторов и партнеров.

### Основные функции
- **Интерактивная презентация**: Многостраничная структура с плавной навигацией.
- **Визуализация данных**: Динамические графики анализа рынка и финансовых прогнозов.
- **Система контактов**: Форма обратной связи с валидацией.
- **Адаптивный дизайн**: Оптимизировано для мобильных устройств.

## Технологический стек

### Frontend
- **React 18** + TypeScript
- **Vite** (сборка)
- **Tailwind CSS** + Shadcn/ui
- **Chart.js** (визуализация графиков)

### Backend
- **Node.js** + Express
- **PostgreSQL** + Drizzle ORM
- **Neon** (серверная часть БД)

## Развертывание
Сайт автоматически развертывается на GitHub Pages при пуше в ветку `main`.
Конфигурация деплоя находится в `.github/workflows/deploy.yml`.
